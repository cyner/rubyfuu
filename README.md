# Rubyfuu

Ruby compiler for Brainfuck programming language. Currently implemented only to
generate 32bit OSX binaries.

## Installation

    $ gem install rubyfuu

## Usage

    $ rubyfuu compile hello_world.b
    $ ./hello_world

...or...

    $ rubyfuu execute hello_world.b
